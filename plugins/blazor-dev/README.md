# blazor-dev

Scaffolding, component generation, and C#/Blazor development guidance for **.NET 8+** projects.

## Agents

| Agent | Description |
|-------|-------------|
| `Blazor Developer` | Expert Blazor + C# agent. Architecture decisions (Server vs WASM vs Auto), component design, state management, DI, async lifecycle guidance. |

## Skills

| Skill | Description |
|-------|-------------|
| `blazor-project-scaffold` | Scaffold a new Blazor project: template selection, folder structure, Program.cs, _Imports.razor, DI setup, authentication, CSS framework integration. |
| `blazor-component-generator` | Generate Blazor components: code-behind pattern, parameters, EventCallback, lifecycle methods, RenderFragment, EditForm, CSS isolation, disposal. |

## Instructions

| File | Applies To | Description |
|------|-----------|-------------|
| `blazor-csharp.instructions.md` | `**/*.{cs,razor}` | Enforces .NET 8+ conventions: file-scoped namespaces, nullable refs, code-behind pattern, component disposal, async lifecycle rules. |
