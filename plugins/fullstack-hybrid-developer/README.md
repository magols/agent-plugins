# fullstack-hybrid-developer

Full-stack .NET plugin for hybrid application development with **ASP.NET Core, EF Core, Blazor, Radzen, Optimizely CMS, Azure, and AI integration**.

## Agents

| Agent | Description |
|-------|-------------|
| `PLUGIN Full-Stack Hybrid Developer` | Expert full-stack hybrid developer mode focused on architecture, implementation, and cloud-native delivery across Microsoft stack technologies. |

## Instructions

| File | Applies To | Description |
|------|-----------|-------------|
| `optimizely-cms.instructions.md` | `**/*.cs, **/*.razor, **/*.cshtml` | Optimizely CMS 12+ patterns: content modeling, controllers/components, DI, content APIs, security, and performance practices. |
| `radzen-blazor.instructions.md` | `**/*.razor, **/*.razor.cs, **/*.cs` | Radzen Blazor component usage patterns, configuration, services, accessibility, and performance guidance. |
